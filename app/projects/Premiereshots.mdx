## Overview

**Premiere Shots** is our signature photography service — and to make it more accessible, I developed a **web booking reservation system** that allows clients to schedule photography sessions online.  

The system combines a clean and intuitive interface with reliable backend functionality, giving users the ability to browse services, book appointments, and manage reservations with ease.

---

## Key Features

- **Online Booking**: Customers can schedule sessions directly through the website.  
- **Service Listings**: Clear presentation of available packages and details.  
- **Responsive Design**: Built with HTML and CSS for compatibility across devices.  
- **Database Integration**: Powered by PHP and MySQL (via XAMPP) for storing bookings and user information.  
- **User-Friendly Flow**: Simplified process from service selection to booking confirmation.  

---

## Technologies Used

- **HTML5 & CSS3** → Layout and responsive design.  
- **JavaScript** → Form validation and interactive UI.  
- **PHP** → Server-side logic for handling bookings.  
- **MySQL (XAMPP)** → Database for users, bookings, and services.  
- **XAMPP** → Local development environment with Apache, PHP, and MySQL.  

---

## Challenges & Learnings

- Ensuring a smooth booking flow without overcomplicating the UI.  
- Setting up **database relationships** in MySQL to manage services, users, and reservations.  
- Validating forms both on the client (JavaScript) and server (PHP) for better data accuracy.  
- Learned how to bridge the gap between **front-end design** and **backend database management**.  

---

## Outcome

The **Premiere Shots Booking System** is now a functioning platform where clients can view services and book sessions online.  

It demonstrates how a straightforward tech stack — HTML, CSS, JavaScript, and PHP with MySQL — can deliver a practical solution that supports a real-world photography business.
